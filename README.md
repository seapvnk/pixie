</p>

## Running pixie
````bash
# clone repo
git clone https://github.com/seapvnk/pixie
cd pixie

# install dependencies
yarn install
# or
npm i

# run
yarn start
# or
npm start

# go to http://localhost:3000 to see the pixie running
````
